# Aula 1 – Conhecendo a Linguagem Python

## Conteúdos
1. **Tipos de dados básicos** – strings, números inteiros, floats, booleanos.
2. **Modo interativo e funções úteis** – dir(), exemplos no terminal.
3. **Variáveis e constantes** – definição, boas práticas, exemplos de listas.
4. **Conversão de tipos (casting)** – int, float, str, bool, diferenças de divisão.
5. **Entrada e saída de dados** – print() e input(), f-strings, parâmetros end e sep.

## Observações
- Cada tópico tem seu próprio arquivo Python.
- Funções organizam os exemplos para execução limpa.
- Os exemplos de `input()` estão comentados para evitar travar a execução do script.
