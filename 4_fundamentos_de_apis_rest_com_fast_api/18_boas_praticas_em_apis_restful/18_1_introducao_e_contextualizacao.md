# Aula 18.1 – Introdução e Contextualização

As APIs RESTful são essenciais para criar sistemas web consistentes, escaláveis e fáceis de manter. Seguir boas práticas ajuda a manter sua API organizada, compreensível e pronta para evolução sem quebrar clientes existentes.

---

## Por que seguir boas práticas?

- Facilita a manutenção do código
- Melhora a experiência de outros desenvolvedores
- Torna a API previsível e consistente
- Evita problemas de segurança e performance
